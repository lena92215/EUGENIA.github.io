★ 大學部畢業專題 VSM / 瓶頸改善

應用價值流程圖於產線檢測瓶頸問題

🔍 發現痛點

產線檢測關卡因工時不均勻，造成半成品堆積與嚴重的人員等待時間，大幅限制整體產能。

⚙️ 執行過程

實地觀測測量各站工時，繪製「現況價值流程圖 (Current VSM)」。運用 IE 七大手法重組工作程序，規劃出「未來狀態圖 (Future VSM)」。

📊 執行成果

成功平衡生產線，消除檢測瓶頸，預估可降低等待浪費，提高產線流動效率。

💿 研究所課堂專案 產品設計 / 問卷分析

環保科技保鮮盒開發計畫

🔍 發現痛點

外食族常面臨加熱不便或保鮮不佳的窘境。市面上缺乏整合「溫控技術」與「攜帶環保」的商品解決方案。

⚙️ 執行過程

設計市場問卷，量化分析消費者習慣與痛點。依數據進行外觀功能規劃，繪製 3D 產品設計圖並剪輯 3D 概念介紹影片。

🎬 執行成果

完成一整套具高可行性的產品開發提案書，包含市場定位、3D設計圖與宣傳影片，成功展示跨領域提案實力。

🦋 研究所課堂專案 ISO 14064 / 減碳模型

企業碳盤查執行計畫書

🔍 執行背景

因應國際減碳法規與綠色供應鏈標準，為本土製造企業規劃高合規、系統化的碳排放量盤查藍圖。

⚙️ 執行過程

依 ISO 14064-1 要求設定盤查邊界，精確設計活動數據（用電、能耗等）收集表單，並建置 Excel 排放係數計算模型。

📈 執行成果

完成包含組織範疇界定、計算架構及節能建議的「企業碳盤查執行報告」，展現卓越的標準法規理解與執行計畫力。
